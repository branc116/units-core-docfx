Regex constants.

HandleBase

Handles lines like

Base(Unit) := Length | Mass | Time

Goal is to handle

Units(Base) := Length | Mass | Time

HandleComposit

Handles lines like:

Velocity := Length / Time

HandleInfer

Handle lines:

Infer

Goal is:

!Infer

HandleMesument

Handle lines:

Unit(Mass) := gram(i, g) | ounce(i*0.03527396, oz)

HandleOperatorDefinition

Handle lines like:

Operator(/) := a = b / c | c = b / a | b = a * c | b = c * a

HandleOperators

Handle lines like:

Operations := (*, Times) | (/, Per)

Goal is to unify all of the operator definitions (see remarks.)

HandleReals

Handle lines like:

Real(Types) := (System.Single, RealFloat) | (System.Double, RealDouble) | (Godot.Vector3, Vec3)

HandleSelfOps

Handle lines like:

SelfOps := (==, Eq, bool) | (!=, Ne, bool) | (+, Plus, null) | (-, Minus, null) | (*, Times, null) | (/, Per, null)

Goal is to unify it with HandleOperators

Interfaces

IHandler

Interface every Handler shuld implement. Handlers job is to provide the System.Text.RegularExpressions.Regex definition for the inputs it can handle MatchRegex(ParserState). When Handler can handle the line, Handle(ParserState, String) will be called by Parse(String[])
