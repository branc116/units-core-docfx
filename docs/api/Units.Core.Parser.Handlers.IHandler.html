Interface IHandler

Interface every Handler shuld implement. Handlers job is to provide the System.Text.RegularExpressions.Regex definition for the inputs it can handle MatchRegex(ParserState). When Handler can handle the line, Handle(ParserState, String) will be called by Parse(String[])
